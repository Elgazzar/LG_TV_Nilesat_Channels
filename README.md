# LG TV Nilesat Channels

This repository contains a curated and organized channel list for LG TVs for the Nilesat satellite.

## Files

- `2026_TV_List.TLL`: The final LG TV channel list file. You can import this directly to your LG TV using a USB flash drive.
- `2026_channels_list.md`: A markdown table listing all the channels included in the `.TLL` file.

## How to Install

1. Format a USB flash drive to FAT32.
2. Copy the `2026_TV_List.TLL` file to the root directory of the USB flash drive.
3. Insert the USB drive into your LG TV.
4. Go to Settings -> Channels -> Copy Channels (or similar depending on your LG TV model).
5. Select "USB to TV" and choose the `.TLL` file.
6. The TV will reboot and the new channel list will be applied.
