# LG TV Nilesat Channels

This repository contains a curated and organized channel list for LG TVs for the Nilesat satellite.

## Files

- `GlobalClone00001.TLL`: The final LG TV channel list file. You can import this directly to your LG TV using a USB flash drive.
- `2026_channels_list.md`: A markdown table listing all the channels included in the `.TLL` file.

## Channel Grouping Layout

The channel list is carefully organized into distinct groups to make navigation easier on your TV. The channels are ordered seamlessly without gaps:
- **Premium & General**: The first block of channels (1 to ~28) contains the most popular general entertainment channels (e.g., ON, dmc, MBC Masr, Sada El Balad, CBC).
- **Series & Drama**: Following the general channels, you will find a dedicated group for TV series and drama (e.g., dmc drama, CBC Drama, Alhayat Muslsalat, ZEE ALWAN, MBC Drama).
- **Movies & Cinema**: Dedicated movie channels (Rotana Cinema, MBC 2, etc.).
- **News, Sports, Kids, & Regional**: Grouped logically further down the list.

## How to Install

1. Format a USB flash drive to FAT32.
2. Copy the `GlobalClone00001.TLL` file to the root directory of the USB flash drive.
3. Insert the USB drive into your LG TV.
4. Go to Settings -> Channels -> Copy Channels (or similar depending on your LG TV model).
5. Select "USB to TV" and choose the `.TLL` file.
6. The TV will reboot and the new channel list will be applied.
